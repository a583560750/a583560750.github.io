在N1小钢炮docker中搭建UnblockNeteaseMusic
发布于：2019-07-16 16:54:22 标签：/ N1 / docker / UnblockNeteaseMusic / 网易云 / 访问：

1、新建镜像

在Web editor中输入以下内容

FROM arm64v8/node:lts-alpine

ENV NODE_ENV production

WORKDIR /usr/src/app

RUN npm init -y

RUN npm install –save https://github.com/nondanee/UnblockNeteaseMusic/tarball/master

EXPOSE 8080

ENTRYPOINT [“npx”, “UnblockNeteaseMusic”]

2、创建部署容器

3、最后配置XX音乐客户端代理重启即可
