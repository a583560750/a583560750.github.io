IPhone 4s 6.1.3成功安装登录微信的方法（至2019.07.08有效）
发布于：2019-07-09 09:52:03 标签：/ 6.1.3 / 微信 / 4s / 访问：

此教程至2019.07.08依然有效

教程为浅蓝月旦制作，博主亲测局部优化修改：

一、准备两部手机：
一部正常登陆微信的任何手机，一部必须iOS8-9的越狱系统

博主只有一台4s与红米note5，本教程以此为例

二、4s升级至ios8.4（有ios8-9其它iPhone手机也行，如本为）

6.1.3未越狱：直接在设置–通用–软件更新中进行更新

6.1.3已越狱：平刷6.1.3为未越狱状态再按照未越狱方式更新（可用爱思助手平刷）

三、4S升级成功后用Impactor_0.9.52工具进行越狱

参考此iOS8.4.1越狱工具EtasonJB正式发布 附详细越狱教程及工具下载 Tips：越狱报错可前往Impactor官网下载最新Impactor工具使用

四、越狱后添加源：apt.25pp.com，然后搜索Apple File Conduit”2”与AppSync Unified插件进行安装

五、用91助手或其他工具安装微信-6.2.0改6.3.1 .ipa，安装后手机不做任意操作（此文件可在文章底部下载）

六、再用91助手或其他工具安装微信微信-6.7.4.44 .ipa，安装后手机不做任意操作（此文件可在文章底部下载）

七、将4S设备变成iPad
用91助手或其他工具找到越狱路劲/User/Library/Caches中的com.apple.MobileGestalt.plist文件，先导出备份，然后搜索iphone，把全部iphone字母替换为iPad，最后保存
八、4S打开微信，然后用其他已登录微信的手机扫描登录

九、登录成功后，退出微信，然后把第7步中备份的com.apple.MobileGestalt.plist文件导入/User/Library/Caches中替换

十、用91助手或其他工具重新安装第5步中的微信-6.2.0改6.3.1 .ipa，安装后手机不做任意操作

十一、用爱思助手在我的设备-应用程序-找到微信-浏览-找到除0000000000外，最长了乱码组合的文件夹打开找到mmsetting.archive文件删除，然后把红包补丁WCRedEnvelopes文件夹，复制到此文件夹（除0000000000外，最长了乱码组合的文件夹）根目录

十二、4S打开成功登入微信，然后使用PP助手2.0（或从2.0升级为最新版的PP助手）或91助手备份微信应用及文档

十三、降级4S为6.1.3并越狱（可用爱思助手直刷6.1.3再用PP助手越狱），然后添加源：apt.25pp.com，搜索安装appsync for ios5.0＋6.0＋插件，最后使用91或爱思助手安装第12步中备份的微信安装包即可
