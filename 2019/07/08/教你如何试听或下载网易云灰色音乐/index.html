教你如何试听或下载网易云灰色音乐
发布于：2019-07-08 12:01:03 标签：/ 音乐 / 网易云 / 访问：

致敬伟大的作者nondanee

原理很简单，其实就是用网易云旧链或其他音乐平台的链接去替换，教程如下：

1、下载源码：UnblockNeteaseMusic
解压zip文件，建议放到 C 盘的目录，不要放到平时安装程序的目录，以免CMD没权限进去导致使用失败。如（C:\UnblockNeteaseMusic-master）

2、安装 Node.js

去官网下载Node.js安装过程中记得勾选Add to Path

3、获得网易云音乐ip：

在 cmd 中输入命令后回车：ping music.163.com获取IP地址

4、新建批处理运行

新建文件，重命名为run.bat，然后编辑如下内容保存：

@echo off
cd C:\UnblockNeteaseMusic-master
node app.js -p 58000 -f 59.111.160.197

注：

第二行中的路径为步骤一存放的路径

第三行中的58000为任意无占用的端口，IP地址为第三步骤中获取的IP

5、右键以管理员运行刚刚保存的run.bat脚本，此批处理不能关闭，关闭即停止服务

6、配置网易云云代理服务后，重启网易云即可

7、配置手机端代理服务（以安卓手机为例），与安装UnblockNeteaseMusic的PC端同一局域网（如下内网穿透可使用FRP、NGROK、花生壳等工具），打开WIFI高级功能，选择手动代理，输入PC的IP地址与UnblockNeteaseMusic运行使用的端口，保存后重启手机版网易云即可
